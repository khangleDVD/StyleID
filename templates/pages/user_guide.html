Hướng dẫn sử dụng

Cách dùng giao diện web {{ site.app_name }} — từ đăng ký đến phân tích ảnh, lịch sử, thống kê và gói cước.

Tài liệu đầy đủ: docs/HUONG_DAN_SU_DUNG.md trong repository.

1. Đăng ký / Đăng nhập

  1. Bấm Đăng ký — nhập tên đăng nhập, mật khẩu, họ tên.
  2. Hoặc bấm Đăng nhập bằng Google (cần quản trị viên cấu hình OAuth).
  3. Sau đăng nhập, sidebar hiển thị tên user và số lượt phân tích còn lại.

Tài khoản username và Google là hai user độc lập — không gộp lịch sử. Tài khoản mới có lượt phân tích miễn phí mặc định.

2. Phân tích ảnh

  1. Vào menu Phân tích ảnh (hoặc bấm «Bắt đầu phân tích» trên trang chủ).
  2. Kéo thả hoặc chọn ảnh (PNG, JPG, JPEG, WEBP).
  3. Có thể chọn nhiều ảnh trong một lần (giới hạn theo cấu hình hệ thống).
  4. Bấm Phân tích — đợi AI xử lý (mỗi ảnh gọi nhiều model vision song song).
  5. Xem kết quả:
    • Danh sách món đồ — tên, danh mục, phong cách, độ tin cậy
    • Phong cách tổng thể — nhãn chính, Top-3, mô tả
    • Dịp phù hợp — gợi ý theo phong cách
  6. Bấm Gợi ý mix (tuỳ chọn) — AI gợi ý thêm món để hoàn thiện outfit.

Mỗi lần bấm Phân tích tiêu tốn một lượt (trừ tài khoản admin).

3. Lịch sử phân tích

  • Menu Lịch sử — xem các lần phân tích trước (cần đăng nhập).
  • Có thể xóa từng bản ghi.
  • Ảnh hiển thị qua đường dẫn /uploads/<tên-file>.

4. Tổng quan (Thống kê)

  • Menu Tổng quan — số lần phân tích, top phong cách, top món đồ.
  • Lọc theo 7 ngày, 30 ngày hoặc toàn bộ.

5. Gói cước

  • Menu Gói cước — xem gói nạp lượt phân tích.
  • Nếu bật thanh toán: quét VietQR, hệ thống đối soát qua SePay và tự cộng lượt.

6. Quản trị (admin)

  • User có role = 'admin' thấy menu Quản trị.
  • Quản lý user, cấp lượt phân tích, xem lịch sử toàn hệ thống.
  • Admin không bị trừ lượt khi phân tích.

7. Ngôn ngữ & giao diện

  • Nút EN / VI trên thanh công cụ — chuyển ngôn ngữ giao diện.
  • Nút chuyển sáng / tối — tuỳ chọn giao diện (lưu trên trình duyệt).

8. Câu hỏi thường gặp

Có cần cài app?
Không — chỉ cần trình duyệt, responsive trên điện thoại.
Ảnh có được lưu không?
Có — trên máy chủ để hiển thị lịch sử. Bạn có thể xóa từng bản ghi.
Làm sao xóa tài khoản?
Gửi email tới {{ site.support_email }}. Xem Chính sách xóa dữ liệu.

User Guide

How to use the {{ site.app_name }} web app — sign-up, image analysis, history, stats, and plans.

Full documentation: docs/HUONG_DAN_SU_DUNG.md in the repository.

1. Sign up / Sign in

  1. Tap Sign up — username, password, display name.
  2. Or Sign in with Google (requires OAuth setup).
  3. After sign-in, the sidebar shows your name and remaining analysis credits.

Username and Google accounts are separate users — history is not merged. New accounts get free credits by default.

2. Image analysis

  1. Open Analyze (or «Start analyzing» on the home page).
  2. Drag & drop or browse images (PNG, JPG, JPEG, WEBP).
  3. You can upload multiple images in one run (system limit applies).
  4. Tap Analyze — wait for AI (parallel vision models per image).
  5. Review results:
    • Item list — name, category, style, confidence
    • Overall style — primary label, Top-3, description
    • Occasions — context suggestions
  6. Optional Mix suggestions — AI suggests items to complete the outfit.

Each Analyze action uses one credit (admin accounts are exempt).

3. History

  • History menu — past analyses (sign-in required).
  • Delete individual records.
  • Images served at /uploads/<filename>.

4. Statistics overview

  • Overview — analysis count, top styles, top items.
  • Filter by 7 days, 30 days, or all time.

5. Service plans

  • Plans — buy analysis credits.
  • If payments enabled: scan VietQR; SePay reconciliation adds credits automatically.

6. Admin

  • Users with role = 'admin' see the Admin menu.
  • Manage users, grant credits, view system-wide history.
  • Admins are not charged for analysis.

7. Language & theme

  • EN / VI toggle — switch UI language.
  • Light / dark theme toggle (saved in browser).

8. FAQ

Need to install an app?
No — browser only, mobile responsive.
Are images stored?
Yes — on the server for history. Delete records individually.
How to delete my account?
Email {{ site.support_email }}. See Data Deletion Policy.
