Hướng dẫn cài đặt

Triển khai {{ site.app_name }} trên máy chủ hoặc máy phát triển — Python, MySQL, OpenRouter và cấu hình .env.

Tài liệu đầy đủ: docs/HUONG_DAN_CAI_DAT.md trong repository.

1. Yêu cầu hệ thống

  • Hệ điều hành: Windows 10/11, macOS hoặc Linux
  • Python: 3.10 trở lên
  • MySQL: 8.0+ (hoặc MariaDB tương thích)
  • RAM: ≥ 8 GB (khuyến nghị khi chạy benchmark)
  • Mạng: Internet để gọi OpenRouter API
  • OpenRouter: tài khoản có credit tại openrouter.ai

2. Cài đặt môi trường

Bước 1 — Mã nguồn

Clone hoặc giải nén dự án, mở terminal tại thư mục gốc.

Bước 2 — Môi trường ảo Python

Windows (PowerShell):

python -m venv venv
.\venv\Scripts\Activate.ps1

Linux / macOS:

python3 -m venv venv
source venv/bin/activate

Bước 3 — Dependency

pip install -r requirements.txt

Bước 4 — File cấu hình

Windows: copy .env.example .env
Linux / macOS: cp .env.example .env

3. Cấu hình .env

Bắt buộc (chạy cơ bản)

  • FLASK_SECRET_KEY — khóa bí mật Flask + JWT
  • MYSQL_HOST, MYSQL_USER, MYSQL_PASSWORD, MYSQL_DATABASE
  • OPENROUTER_API_KEY — API key OpenRouter (sk-or-v1-...)

Không có OPENROUTER_API_KEY → hệ thống vẫn chạy nhưng trả kết quả demo, không phân tích AI thật.

AI — phân tích ảnh (tuỳ chọn)

  • OPENROUTER_VISION_MODELS — 3 model vision, cách nhau dấu phẩy
  • OPENROUTER_MAX_TOKENS — mặc định 512
  • MAX_ANALYZE_IMAGES — tối đa ảnh mỗi lần upload (mặc định 5)

Google OAuth & thanh toán (tuỳ chọn)

Xem comment trong .env.example cho GOOGLE_*, VIETQR_*, SEPAY_*.

4. Khởi tạo cơ sở dữ liệu

mysql -u root -p < scripts/init_mysql.sql

Script tạo database, bảng users, history, payments.

Sau khi đăng ký tài khoản đầu tiên, gán quyền admin (khuyến nghị):

USE lumistyle_db;
UPDATE users SET role = 'admin' WHERE id = 1;

Admin không bị trừ lượt phân tích và truy cập trang Quản trị.

5. Chạy ứng dụng

Đảm bảo MySQL đang chạy, rồi:

python app.py

Mặc định: http://localhost:5000

  • Trang chủ load được → giao diện StyleID hiển thị
  • Đăng ký / đăng nhập → nhận JWT
  • Upload ảnh (có API key) → kết quả sau ~30–90 giây/ảnh

6. Xử lý sự cố thường gặp

MySQL không kết nối được
Kiểm tra service MySQL, biến MYSQL_* trong .env, chạy lại scripts/init_mysql.sql nếu DB chưa tồn tại.
Phân tích trả kết quả demo
Thiếu hoặc sai OPENROUTER_API_KEY — sửa .env và restart python app.py.
Lỗi 402 / hết credit OpenRouter
Nạp credit tại openrouter.ai hoặc giảm OPENROUTER_MAX_TOKENS.
Không đủ lượt phân tích
Dùng tài khoản admin hoặc nhờ admin cấp thêm lượt trên trang Quản trị.

Installation Guide

Deploy {{ site.app_name }} on a server or dev machine — Python, MySQL, OpenRouter, and .env setup.

Full documentation: docs/HUONG_DAN_CAI_DAT.md in the repository.

1. System requirements

  • OS: Windows 10/11, macOS or Linux
  • Python: 3.10+
  • MySQL: 8.0+ (or compatible MariaDB)
  • RAM: ≥ 8 GB (recommended for benchmarks)
  • Network: Internet for OpenRouter API
  • OpenRouter: account with credits at openrouter.ai

2. Environment setup

Step 1 — Source code

Clone or extract the project, open a terminal at the root folder.

Step 2 — Python virtual environment

Windows (PowerShell):

python -m venv venv
.\venv\Scripts\Activate.ps1

Linux / macOS:

python3 -m venv venv
source venv/bin/activate

Step 3 — Dependencies

pip install -r requirements.txt

Step 4 — Configuration file

Windows: copy .env.example .env
Linux / macOS: cp .env.example .env

3. .env configuration

Required (basic run)

  • FLASK_SECRET_KEY — Flask secret + JWT signing
  • MYSQL_HOST, MYSQL_USER, MYSQL_PASSWORD, MYSQL_DATABASE
  • OPENROUTER_API_KEY — OpenRouter API key

Without OPENROUTER_API_KEY, the app runs but returns demo results only.

AI — image analysis (optional)

  • OPENROUTER_VISION_MODELS — comma-separated vision models
  • OPENROUTER_MAX_TOKENS — default 512
  • MAX_ANALYZE_IMAGES — max images per upload (default 5)

Google OAuth & payments (optional)

See .env.example for GOOGLE_*, VIETQR_*, SEPAY_*.

4. Database initialization

mysql -u root -p < scripts/init_mysql.sql

Creates database and users, history, payments tables.

After first sign-up, grant admin (recommended):

USE lumistyle_db;
UPDATE users SET role = 'admin' WHERE id = 1;

Admins are not charged analysis credits and can access the Admin page.

5. Run the application

Ensure MySQL is running, then:

python app.py

Default URL: http://localhost:5000

  • Home page loads → StyleID UI appears
  • Sign up / sign in → JWT issued
  • Upload images (with API key) → results in ~30–90s per image

6. Troubleshooting

Cannot connect to MySQL
Check MySQL service, MYSQL_* in .env, re-run scripts/init_mysql.sql if DB missing.
Analysis returns demo results
Missing or invalid OPENROUTER_API_KEY — fix .env and restart python app.py.
402 / out of OpenRouter credits
Add credits at openrouter.ai or lower OPENROUTER_MAX_TOKENS.
Not enough analysis credits
Use an admin account or ask an admin to grant credits.
